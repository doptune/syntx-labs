    SYNTX LABS / DOCUMENTATION

    Syntx Labs

    Local AI training, workflow orchestration, automation runtime, and ontology-driven execution environment.

    Version 1.0 MIT Licensed Doptune
    OVERVIEW

    Syntx Labs is a fully local AI execution environment designed for ontology-based knowledge training, deterministic skill execution, workflow orchestration, and local automation pipelines.

    The platform combines a Tauri desktop shell, a Python execution backend, local Ollama-hosted models, and SQLite-backed knowledge structures into a unified runtime architecture.

    ARCHITECTURE
    Tauri (Rust)
  ├── Windowing
  ├── IPC
  ├── Filesystem Access
  └── Process Spawning

Frontend (HTML/CSS/JS)
  ├── Knowledge Ontology
  ├── Instruction Console
  ├── Validation Environment
  ├── Skill Constellation
  └── Workflow Orchestrator

Python Runtime
  ├── Skill Executor
  ├── Workflow Executor
  ├── Capability Checker
  ├── Action Library
  └── Parser Layer

Persistence Layer
  └── SQLite Ontology Databases
    CORE COMPONENTS
